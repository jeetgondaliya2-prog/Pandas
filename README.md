  # 🐼 Pandas Practice Repository

This repository contains my hands-on practice and learning exercises using the **Pandas** library in Python. It includes Jupyter notebooks covering data manipulation, analysis, cleaning, filtering, grouping, and visualization basics.

## 📂 Repository Contents

- `Ep-1.ipynb` – Introduction to Pandas
- `Ep-2.ipynb` – Series and DataFrames
- `EP-3.ipynb` – Data Selection & Indexing
- `Ep-4.ipynb` – Data Cleaning
- `Ep-5.ipynb` – Handling Missing Values
- `Ep-6.ipynb` – Sorting & Filtering
- `Ep-7.ipynb` – GroupBy Operations
- `Ep-8.ipynb` – Merging & Joining DataFrames
- `Ep-9.ipynb` – Advanced Pandas Operations
- `IPL Analysis.ipynb` – IPL Dataset Analysis
- `matches.csv` – IPL Matches Dataset
- `deliveries.csv` – IPL Deliveries Dataset

## 📦 Installation

Clone this repository:

```bash
git clone https://github.com/jeetgondaliya2-prog/Pandas.git
```

Move into the project folder:

```bash
cd Pandas
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib jupyter
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
